    Settings

    離開前詢問：

    Confirm before exiting. 在離開前確認，注意，瀏覽器可能基於效能原因，而沒有在離開前詢問。

    Change the bacground. 更換背景色，支援CSS語法，例如：black。詳細調色盤可以在 http://www.w3schools.com/colors/colors_picker.asp 進行調色。

    顯示渦碼：

    Display the code of Raid. 顯示渦碼，預設為否，由於渦碼更新過慢效果不佳，因此此功能預設為關閉，未來將視情形將此功能停用。

    關燈：

    Darken the page. 可以讓主畫面變得不那麼顯眼，降地了畫面的明亮度，和增加透明度，可以用在不想被別人發現，或是晚上關燈時用，這個效果不會被保存到下次網頁載入。

    阻止網頁重新整理：

    Avoid from refreshing the page. 這個功能用在畫面顯示斷線時使用，可以阻止網頁被重新整理，如果斷線為誤判， 則還可以繼續任務或對戰，但是IE不支援，詳細請看 http://www.w3schools.com/jsref/met_win_stop.asp 的 Browswer Support 章節，這個效果不會保存到下次網頁載入。部分機型可能產生高CPU，請不要長時間開啟。

    Shorten URL. 這個功能適合用在轉貼和 Unlight 的相關網址，可以得到例如：https://unlight.cc/s/zgJpVO 的結果。
